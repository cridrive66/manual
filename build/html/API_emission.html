API Emission

The API Emission provides a set of tools where the user can calculate the load of certain APIs at different WWTPs, transfer it to the river network, calculate the concentration and perform the risk assessment in each river section within a catchment.

This group of tools is divided in 4 different processes:

  • Consumption Selection
  • Emission Loads
  • Accumulation
  • Risk Assessment

Each tool is documented in the following sections with explanations of its functionality and step-by-step tutorials.
